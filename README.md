# Insurance Policy Management API

A RESTful API for managing insurance policies, built with Java and Spring Boot.

## 🛠️ Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- SQL (MySQL/PostgreSQL)
- Maven

## ✨ Features
- Create, read, update, and delete insurance policies
- Policy holder management
- Input validation
- Centralized exception handling
- Pagination and sorting support

## 📂 Project Structure

    src/main/java/com/srijan/policyapi/
    ├── controller/    # REST endpoints
    ├── service/       # Business logic
    ├── repository/    # Data access layer
    ├── entity/        # JPA entities
    ├── dto/           # Data transfer objects
    └── exception/     # Custom exceptions & handlers
